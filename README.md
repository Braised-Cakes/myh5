# myh5

> myh5页面制作工具, 一键生成H5酷炫页面, 类似于易企秀!

## 技术栈

vue, node, sass, express, mongoose


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run start

# build for production with minification
npm run build

```
