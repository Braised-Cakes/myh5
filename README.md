# my_h5

> myh5页面制作工具,微信营销推广专业场景制作平台,企业宣传、音乐相册、电子贺卡、邀请函、微杂志等一键生成H5酷炫页面,易企秀为您助力移动自主营销!

## 技术栈

vue, node, sass, mongoose


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run start

# build for production with minification
npm run build

```
